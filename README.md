> TRON / USDT 钱包靓号 GPU 版本  
> 支持：高仿地址、连号地址、自定义前后缀、顺子号、6/7/8/9/10A 等多种靓号形态  
> 推荐搭配独立显卡 / GPU 服务器使用，支持断网纯本地运行（更安心）

## 项目简介

本项目是一个基于 GPU 加速的 **波场 TRON 钱包靓号生成器**，  
适用于：

- USDT 钱包靓号生成  
- GPU 波场靓号批量爆破  
- 高仿地址（前 N 位 / 后 N 位）生成  
- AA 连号 / 顺子号 / 自定义前后缀靓号  

支持在 **虚拟机、物理服务器、云电脑** 等环境运行，绿色单文件，不需安装复杂依赖，  
可以直接断网使用，仅在本地生成私钥与地址。

> ⚠️ 本工具仅供技术研究与学习，不建议用于任何违法或高风险场景。  
> 生成的钱包 *务必自行做好私钥备份与安全管理*。

## 功能特性

- 🚀 **GPU 并行加速**  
  - 支持多线程 + GPU CUDA / OpenCL 内核  
  - 单张 2060 实测约 1 秒 ~ 135 MH/s（约每秒 1 亿 3500 万个地址）

- 🎯 **多种匹配模式**  
  - 前缀 / 后缀匹配（例：前 5 位 TTRX8，后 6 位 888888）  
  - 连号地址：6A/7A/8A/9A/10A 等  
  - 高仿地址：按目标地址的前 N 位 + 后 N 位去“撞库”  
  - 顺子地址：123456 / 234567 / ABCDEF 等连续字符  
  - 自定义模式：支持简单通配 / 正则（视版本配置）

- 💾 **断点续跑 & 自动分文件保存**  
  - 定期保存进度  
  - 命中靓号自动按类型写入不同 txt 文件  
  - 支持运行结束后再次启动继续跑

- 📦 **批量导出信息**  
  - TRON 地址（T 开头）  
  - 公钥、公钥哈希  
  - 私钥（HEX / WIF 可选）  
  - 生成时的参数记录，方便二次校验

- 🧰 **跨平台支持**  
  - CPU 模式：Windows / Linux / macOS  
  - GPU 模式：NVIDIA / AMD / Intel（视驱动提供的 OpenCL / CUDA 支持）

## 工作原理速览（TRON 靓号）

1. 随机遍历 secp256k1 私钥空间，生成随机私钥 `sk`；  
2. 用 `sk` 计算椭圆曲线公钥 `pk`；  
3. 对 `pk` 做 `keccak256`，取后 20 字节作为核心地址主体；  
4. 在前面拼接波场主网标识 `0x41` 得到 21 字节地址；  
5. 对这 21 字节执行 Base58Check（双 SHA256 校验加校验位），  
   输出以 `T` 开头的人类可读 TRON 地址；  
6. 根据你设定的匹配规则（前缀 / 后缀 / 连号 / 高仿 / 顺子）做判断：  
   - 若命中 → 写入对应的结果文件  
   - 若不命中 → 继续下一个随机私钥

## 文件说明

- `gd_gpu.exe`  
  主程序（示例名称，以实际发布二进制为准）

- `config.txt`  
  - 前几行数字为“基数/种子”参数，用于调整随机分布和连号出号概率  
  - **你无需理解具体数学含义**，只需偶尔改一改，重新启动程序即可换一套随机序列  
  - 新版本已默认优化好 9A / 10A 等高难度连号的出号难度

- `address.txt`  
  - 用于生成 **高仿地址** 的列表  
  - 一行一个目标地址（真实地址），程序会尝试生成“前 N 位 + 后 N 位”相同的高仿地址  
  - 默认仿前 3 位、后 4 位，你可以在 bat 或参数中自行调节

- `diy.txt`（可选）  
  - 自定义靓号模式  
  - 可以填写 **前 10 位 + 后 10 位** 的模式  
  - 例如：  
    - 只跑后 10 位，就把前 10 位写 `T`（占位），后 10 位写你想要的尾数  
    - 或者前后都写实际字符，跑“指定前后缀的地址”

- `点击运行.bat / 高仿地址.bat`  
  - Windows 一键启动脚本  
  - 内部是对 `gd_gpu.exe` 的命令行封装，你可以右键“编辑”自行修改参数：

  示例：
  ```bat
  gd_gpu.exe --lianhao 6 --down 123.txt
